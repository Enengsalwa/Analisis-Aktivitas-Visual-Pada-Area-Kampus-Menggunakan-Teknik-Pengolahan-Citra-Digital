Mata Kuliah      : Pengolahan Citra Digital
Dosen pengampu   : M. Ikhsan Thohir, M.Kom


Perkembangan teknologi digital menjadikan citra digital sebagai media penting dalam penyampaian informasi visual, termasuk untuk analisis aktivitas di lingkungan kampus yang memiliki kondisi visual kompleks. Namun citra yang diperoleh secara langsung sering mengalami kendala seperti pencahayaan tidak merata, noise, dan kontras rendah sehingga perlu pengolahan citra untuk meningkatkan kualitasnya. 

Tahapan pengolahan citra seperti histogram equalization untuk peningkatan kontras, filtering untuk reduksi noise, deteksi tepi dan morfologi untuk penajaman struktur objek, serta segmentasi seperti Otsu Thresholding untuk pemisahan objek dari latar belakang dapat membantu menghasilkan citra yang lebih jelas dan informatif. 
Melalui penerapan tahapan tersebut, citra kampus dapat dianalisis secara visual dengan lebih akurat untuk mengamati objek maupun aktivitas secara nyata.
